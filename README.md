✈️ Прогнозирование цены авиабилета



Streamlit-приложение для предсказания стоимости авиабилетов на основе модели CatBoost.  

Проект включает сбор данных через API Travelpayouts, обучение модели и удобный веб-интерфейс.

Функциональность



\- Предсказание цены авиабилета по введённым параметрам: города вылета и назначения, дата вылета, количество пересадок.

\- Автоматический расчёт расстояния между аэропортами по их координатам (если маршрут не встречался в исторических данных).

\- Оценка длительности полёта на основе расстояния.

\- Отображение входных параметров и средней абсолютной ошибки модели.



Технологии



\- Python 3.9+

\- Streamlit – интерактивный веб-интерфейс

\- CatBoost – градиентный бустинг для регрессии

\- Pandas, NumPy – обработка данных

\- Scikit-learn – метрики и разбиение выборки

\- Requests – сбор данных через API



Установка и запуск



1\. Клонируйте репозиторий

```bash

git clone https://github.com/opikinss/flight-price-predictor.git

cd flight-price-predictor

2\. Создайте и активируйте виртуальное окружение (рекомендуется)

python -m venv venv

\# Для Windows:

venv\\Scripts\\activate

\# Для Linux/macOS:

source venv/bin/activate

3\. Установите зависимости

pip install -r requirements.txt

4\. Запуск

streamlit run app.py
